# QwenMine

A Minesweeper clone built with C++ and Qt6.

## Build

```bash
mkdir build && cd build
cmake ..
make
```

## Features

- Classic Minesweeper gameplay
- Flag mines with right-click (not yet implemented)
- Flood fill for empty cells
- Win/lose detection
- Mine counter

## Controls

- Left-click: Reveal cell
- (Right-click: Flag mine - not yet implemented)

## Running

```bash
./build/QwenMine
```
